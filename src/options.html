Version

Manifest V3 🎉

  • Added sorting option for a bookmark moved to a folder(put at the top of the list)
  • Added the ability to move the bookmark to the parent folder (if the option of the button to return to the parent folder is enabled)
  • Added quick back to top button
  • Sort parameter changes:
    • Sort by date
    • Sort by folder type (folders at the top of the list, folders at the bottom of the list)
    Note: if you had the old option enabled "Sort by newest", you will need to select in the settings in the list "By date". If one of the sort items is selected, drag and drop sorting will be disabled
  • Favicons from google instead of browser favicon api
  • UI improvements
